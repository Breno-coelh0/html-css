Principais formatações

Negrito

Temos 2 formas de botar uma palavra em negrito

1- forma não semantica:

usando a tag "b" para botar essa palavra em negrito

2- forma semantica:

usando a tag "strong" para botar essa palavra em negrito

Itálico

forma não semantica:

usando a tag "I" para botar essa palavra em itálico

forma semantica:

usando a tag "em" para botar essa palavra em italico
